# Liquidation Nord - E-commerce Platform

Full-stack e-commerce platform for furniture liquidation based in Bondues, France.

## Features

- **Product Management**: Bulk XLSX upload, photo management, inventory tracking
- **Shopping Experience**: Cart, checkout, multiple delivery options
- **Delivery Integration**: Mondial Relay, Cocolis, pickup
- **Admin Dashboard**: Upload products, manage orders, analytics
- **Cross-platform**: Links to Leboncoin for additional sales

## Tech Stack

- **Frontend**: Next.js 15, TypeScript, Tailwind CSS
- **Database**: MongoDB Atlas
- **Payments**: Stripe
- **File Upload**: Uploadthing
- **Delivery APIs**: Mondial Relay SDK, Cocolis API
